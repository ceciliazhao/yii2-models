Class IgnorableBlameableBehavior

IgnorableBlameableBehavior automatically fills the specified attributes with the current user ID, and it is allowed to skip specifying attribute which specified.

To use IgnorableBlameableBehavior, insert the following code to your ActiveRecord class:

use vistart\Models\behaviors\IgnorableBlameableBehavior;

public function behaviors()
{ return [ IgnorableBlameableBehavior::className(), ];
}

By default, IgnorableBlameableBehavior will fill the created_by and updated_by attributes with the current user ID when the associated AR object is being inserted; it will fill the updated_by attribute with the current user ID when the AR object is being updated. If your attribute names are different, you may configure the [[createdByAttribute]] and [[updatedByAttribute]] properties like the following:

public function behaviors()
{ return [ [ 'class' => IgnorableBlameableBehavior::className(), 'createdByAttribute' => 'author_id', 'updatedByAttribute' => 'updater_id', ], ];
}
