Trait BlameableTrait

该 Trait 用于处理属于用户的实例。包括以下功能： 1.单列内容；多列内容待定； 2.内容类型；具体类型应当自定义； 3.内容规则；自动生成； 4.归属用户 GUID； 5.创建用户 GUID； 6.上次更新用户 GUID； 7.确认功能由 ConfirmationTrait 提供； 8.实例功能由 EntityTrait 提供。
