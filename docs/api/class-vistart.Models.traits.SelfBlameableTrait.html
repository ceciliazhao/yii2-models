Trait SelfBlameableTrait

This trait is designed for the model who contains parent. The BlameableTrait use this trait by default. If you want to use this trait into seperate model, please call the initSelfBlameableEvents() method in init() method, like following:

public function init()
{ $this->initSelfBlameableEvents();  // put it before parent call. parent::init();
}
