Trait UserTrait

Assemble PasswordTrait, RegistrationTrait and IdentityTrait into UserTrait. This trait can only be used in the class extended from [[BaseEntityModel]], [[BaseMongoEntityModel]], [[BaseRedisEntityModel]], or any other classes used [[EntityTrait]]. This trait implements two methods create() and findOneOrCreate(). Please read the notes of methods and used traits for further detailed usage.

vistart\Models\traits\UserTrait uses vistart\Models\traits\PasswordTrait, vistart\Models\traits\RegistrationTrait, vistart\Models\traits\IdentityTrait
